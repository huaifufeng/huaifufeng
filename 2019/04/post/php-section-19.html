控制结构

if else

if语句允许当满足条件时，执行一段代码。当你想条件不满足时执行另外的代码可以使用if else。

if ($a > $b) {
    echo "a is greater than b";
} else {
    echo "a is not greater than b";
}

查看PHP手册。

三元运算符是if-else的短语法

三元预算符根据条件是true还是false来执行一些条件。这是一个比较预算符，通常用来将if-else语句变成简单的格式。这允许快速测试一个条件，然后通常用来替换多行的if语句，让你的代码更加紧凑。

下面就是一个三元运算符的例子，这里$a=1,$b=2:

echo ($a > $b) ? "a is greated than b" : "a is not greater than b";

输出：a is not greater than b

控制结构备用语法

PHP提供了控制结构备用的语法：if while for foreach switch。

和普通的语法相比，不同的地方在于：这里使用:来代替开始花括号，使用endif; endwhile; endfor; endforeach; endswitch;来代替结束花括号。有关各个示例，请参阅有关控制结构的备用语法的主题。

if ($a == 42):
    echo "The answer to life, the universe and everything is 42.";
endif;

多个elseif语句使用短语法：

if ($a == 5):
    echo "a equals 5";
elseif ($a == 6):
    echo "a equals 6";
else:
    echo "a is neither 5 nor 6";
endif;

PHP手册：控制结构备用语法。

while

当指定的条件为真时，就循环处理代码块。

$i = 1;
while ($i < 10) {
    echo $i;
    $i++; 
}

将会输出：

123456789

更详细的信息，查看循环那部分内容。

do-while

do-while首先执行一遍代码块，然后根据指定条件是不是true来决定是否执行：

$i = 0; do {
$i++;
    echo $i;
} while ($i < 10);
Output: `12345678910`

goto

PHP5.3开始提供了goto操作符，允许跳转到程序另外的部分。

goto的使用方法就是goto MyLabel。

想要跳转的地方，需要使用MyLabel标记。

下面的例子会输出Hello World!。

<?php
goto MyLabel:
echo "This text will be skipped, because of the jump.";

MyLabel:
echo 'Hello World!';

declare

declare用于为代码块设置执行指令。

下面是有效的指令：

  • ticks
  • encoding
  • strict_types

例如，设置ticks为1：

declare(ticks=1);

想要启动严格模式，就使用declare语句开启严格模式strict_types。

declare(strict_types);

include&require

require

require和include是相似的，除了它会产生一个E_COMPLIE_ERROR级别的错误。当require失败的时候，将会中断脚本的执行。当include失败的时候，不会中断脚本的执行，会报一个E_WARING级别的错误。

require 'file.php';

PHP手册：require

include

include包含并执行一个文件。

./variables.php

$a = 'Hello World!';

./main.php

include 'variables.php';
echo $a;
//输出：Hello World

要小心这个，因为这将造成代码异味。因为包含文件将会改变变量范围内的变量数量和内容。

你也可以包含返回值的文件。这在处理配置文件时非常有用。

configuration.php

<?php
return [
    'dbname' => 'my db',
    'user' => 'admin',
    'pass' => 'password'
];

main.php

<?php
$config = include "configuration.php";

这种方式就会避免包含文件引入变量或修改变量。

PHP手册：include

如果文件返回一个值的话，require & include也可以用来给变量赋值。例如:

include1.php

<?php  
    $a = "This is to be returned";
    return $a;
?>

index.php

 $value = include 'include1.php';
// Here, $value = "This is to be returned"

return

return语句返回程序的控制给调用函数。

当函数里面使用了return语句，这个函数就结束了。

function returnEndsFunctions() {
   echo 'This is executed';
   return;
   echo 'This is not executed.';
}

当你执行这个函数的时候，将会输出This is executed。

当函数内部调用return，并且带有参数时，这个函数的执行就结束了，并且将参数返回给调用函数。

for

for循环用来循环执行一段代码一定次数，这时候非常有用。

for ($i = 1; $i < 10; $i++) {
    echo $i;
}
//输出 123456789

foreach

foreach是语言结构，这可以让你循环数组和对象很简单。

$array = [1, 2, 3];
foreach ($array as $value) {
    echo $value;
}

//输出： 1 2 3

想要对一个对象使用foreach函数，这个对象必须首先Iterator接口。

$array = ['color'=>'red'];
foreach($array as $key => $value){
    echo $key . ': ' . $value;
}

//输出 color: red

if elseif else

elseif

elseif是if和else的组合。if语句被扩展在不满足原本if语句的情况下执行不同的语句。但是只有在满足elseif语句的条件时才会执行。

下面的例子会按照条件，返回其中的一个：

if ($a > $b) {
    echo "a is bigger than b";
} elseif ($a == $b) {
    echo "a is equal to b";
} else {
    echo "a is smaller than b";
}

多个elseif语句

一个if语句，你可以使用多个elseif语句：

if ($a == 1) {
    echo "a is One";
} elseif ($a == 2) {
    echo "a is Two";
} elseif ($a == 3) {
    echo "a is Three";
} else {
    echo "a is not One, not Two nor Three";
}

if

if语句允许你有条件的执行代码片段：

if ($a > $b) {
  echo "a is bigger than b";
}

PHP手册：if

switch

switch可以用更少的代码执行和一系列if语句相同的功能。

switch语句中值将会被测试，和每一个case语句的值进行比较，匹配到的case语句的代码将会被执行。如果没有匹配到结果，并且default语句存在，就会执行default语句。

每一个case语句和default语句都要使用break结束。这将会中断switch语句的执行，跳过后面的代码。如果break语句被忽略了，下一个case语句将会被执行，哪怕没有匹配。忘记break将会造成不希望的代码的执行，当然这种多个case匹配时很有用。

switch ($colour) {
case "red":
    echo "the colour is red";
    break;
case "green":
case "blue":
    echo "the colour is green or blue";
    break;
case "yellow":
echo "the colour is yellow";
// note missing break, the next block will also be executed
case "black":
    echo "the colour is black";
    break;
default:
    echo "the colour is something else";
    break;
}

处理比较固定的值，也可以比较动态的语句(在switch语句提供布尔值，语句表达式返回布尔值)。注意只有第一个匹配的case会输出，所以下面会输出more than 100：

$i = 1048; 
switch (true) { 
case ($i > 0):
    echo "more than 0";
    break;
case ($i > 100):
    echo "more than 100";
    break;
case ($i > 1000):
    echo "more than 1000";
    break;
}

这个语句使用弱类型比较，这可能造成问题。查看Switch警告部分。

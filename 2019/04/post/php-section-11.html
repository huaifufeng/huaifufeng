引用

引用赋值

这是引用使用的第一个阶段。实际你在引用赋值的时候，这里允许你两个变量拥有相同的值。

$foo = &$bar;

$foo和$bar是相等的。它们指向相同的地址，而不是不同的地址。

你也可以在array()语言结构中使用引用赋值。并不是严格的引用赋值：

$foo = 'hi';
$bar = array(1, 2);
$array = array(&$foo, &$bar[0]);

**注意：**数组里面的引用是非常危险的一种行为。使用右侧引用对左侧进行普通赋值不会将左边变为引用。但是数组中这样的赋值引用将会被保留。在函数中，数组作为值传递时也有这个问题。

引用赋值不限制在变量和数组中。在函数和所有"引用传递的值"都可以。

function incrementArray(&$arr) {
    foreach ($arr as $$val) {
        $var++;
    }
}

function &getArray() {
    statice $arr = [1, 2, 3];
    return $arr;
}

incremetnArray(getArray());
var_dump(getArray());  //[2,3,4]

赋值是上面函数定义的关键点。不能通过引用传递表达式，只能传递值/变量。

返回引用

偶尔会出现让你通过引用隐藏返回的情况。

引用返回在函数定义的参数是引用变量的时候非常有用。不要用返回引用来提升性能。PHP引擎会自动处理这些问题。只有在您有有效的技术原因时才返回引用。

查看返回引用的PHP文档。

有很多不同的方式可以返回引用，包括下面的例子：

function parent(&$var) {
    echo $var;
    $var = "updated";
}

function &child() {
    static $a = "test";
    return $a;
}

parent(child());  //返回test
parent(child());  //返回updated

引用返回不仅限于函数引用。您还可以隐式调用函数：

function &myFunction() { 
    static $a = 'foo'; 
    return $a;
}

$bar = &myFunction();
$bar = "updated";
echo myFunction();

你不能直接引用函数的调用，必须在使用之前赋值给一个变量。想要看发生了什么，简单的echo &myFunction();就可以了。

注意：

  • 你被要求在定义和使用的地方都使用&。这个意思是在function &myFunction(){…}函数定义和function callFuncton(&$variable) { … or &myFunction();}都是这样。
  • 你可以只返回一个变量的引用。像上面例子中的$a。因此你不能反回表达式，否则就会产生一个E_NOTICE的PHP错误Notice: Only variable references should be returned by reference in ……
  • 引用返回确实有合法的使用方式。但是我还是要说明：最好在其他方式都有问题的情况下在谨慎使用这个。

传递引用

这允许你传递一个变量的引用给函数或一个元素，这允许你修改变量原本的值。

传递引用不限制在变量，下面这些类型也可以使用：

  • new语句，例如foo(new SomeClass)
  • 返回引用的函数

数组

传递引用常见的用法是修改一个数组的值而不用创建新的数组或者丢弃命名空间。通过引用传递就是在变量的前面增加& ，如果&$myElement。

下面是利用数组中的元素并简单地将1添加到其初始值的示例。

$arr = array(1, 2, 3, 4, 5);

foreach($arr as &$num) {
    $num++;
}

现在当你开始使用这个数组$arr的元素时，通过引用传递的值已经被增加了1。你可以使用下面的代码验证：

print_r($arr);

注意

你在循环中使用引用的时候要小心注意。在循环的最后，$num的值是数组最后一个元素的引用值。在循环之外使用这个值，将会操作数组的最后一个值。你可以在循环结束之后使用unset函数处理这个变量。

$myArray = array(1, 2, 3, 4, 5);
foreach($myArray as &$num) {
   $num++;
}
unset($num);

这将避免出现任何问题。一个出问题的例子可以查看Stack Overflow的问题。

函数

另一个常用的方式是传递引用参数给函数。修改原始的值像下面这样简单：

$var = 5;
// define
function add(&$var) {
    $var++; 
}
// call
add($var);

下面将会输出$var的值：

echo $var;  //6

在函数使用时有一些限制，下面就是PHP文档指出的限制：

在函数调用的时候不需要&符，只有在定义的时候需要。只在函数定义使用已经足够引用传递了。在PHP5.3.0，你使用foo(&$a)将会获取一个call-time pass-by-reference的不推荐信息。在PHP5.4.0已经移除了调用时引用，并且会出现一个致命错误。

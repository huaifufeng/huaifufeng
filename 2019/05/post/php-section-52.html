对象序列化

1. serialize / unserialize

serialize()返回一个包含PHP中存储的任何值的字节流格式的字符串。unserialize()将会把这个变量转换为初始的值。

序列化对象

serialize($object);

反序列化对象

unserialize($object)

例子

$array = array();
$array["a"] = "Foo";
$array["b"] = "Bar";
$array["c"] = "Baz";
$array["d"] = "Wom";

$serializedArray = serialize($array);
echo $serializedArray; //output: 
a:4:{s:1:"a";s:3:"Foo";s:1:"b";s:3:"Bar";s:1:"c";s:3:"Baz";s:1:"d";s:3:"Wom";}

2. Serializable接口

简介

实现这个接口的类不在支持__sleep()和wakeup()魔术方法。当实例序列化的时候，serialize方法将会调用。除非在方法内部编程，否则这将不会调用__destruct()和其他有副作用的方法。当类的数据要反序列化的时候unserialize()方法将会作为构造函数调用，而不是__construct()。如果你想要执行标准的构造方法，就在这个方法里面调用。

基本用法

class obj implents Serializable {
    private $data;
    public function __construct() {
        $this->data = "My private data";
    }
    
    public function serialize() {
        return serialize($this->data);
    }
    
    public function unserialize($data) {
        $this->data = unserialize($data);
    }
    
    public function getData() {
        return $this->data;
    }
}

$obj = new Obj();
$user = serialize($obj);

var_dump($user); //string(38) "C:3:"obj":23:{s:15:"My private data";}"

$newObj = unserialize($user);
var_dump($newobj->getData()); // string(15) "My private data"

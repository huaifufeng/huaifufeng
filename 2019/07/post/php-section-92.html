常见错误

1. call fetch_assoc on boolean

如果你获得一个类似下面的错误：

Fatal error: Call to a member function fetch_assoc() on boolean in C:\xampp\htdocs\stack\index.php on line 7

其他类似的错误包括：

mysql_fetch_assoc() expects parameter 1 to be resource, boolean given...

这些错误意味着要么你的查询有错误(这是一个PHP/Mysql错误)或引用错误。下面的代码就可以产生上面的错误：

$mysqli = new mysqli("localhost", "root", "");

$query = "SELCT * FROM db"; //注意这里的错误
$result = $mysqli->query($query);

$row = $result->fetch_assoc();

为了”修正”这个错误，推荐设置mysql抛出异常：

//在脚本的开头添加下面的内容
mysqli_report(MYSQLI_REPORT_ERROR | MYSQLI_REPORT_STRICT);

这将会抛出一个带有更加详细的异常信息：

You have an error in your SQL syntax; check the manual that corresponds to your MariaDB server version for the right syntax to use near 'SELCT * FROM db' at line 1

另一个会产生相似错误的是你只是简单的将错误信息提供给mysql_fetch_assoc方法或相似的：

$john = true;
mysqli_fetch_assoc($john, $mysqli);

2. Unexpected $end

Parse error: syntax error, unexpected end of file in C:\xampp\htdocs\stack\index.php on line 4

如果你获取类似这个的错误(或者类似unexpected $edn，根据PHP的版本)，你需要确保你已经匹配了所有的反逗号，所有的括号，所有的大括号，所有的方括号等等。

下面的代码会产生上面的错误：

<?php
if (true) {
    echo "asdf";
?>

注意缺少的花括号。注意错误中显示的行号信息 - 这总会显示温度的最后一行。

在终端提示显示git的分支名称

你可以在PS1的变量里面使用函数，只要你确保使用单引号包含这个值或者对于特殊的字符使用逃脱符号。

gitPS1(){
    gitps1=$(git branch 2>/dev/null | grep '*') 
    gitps1="${gitps1:+ (${gitps1/#\* /})}"
    echo "$gitps1"
}

PS1='\u@\h:\w$(gitPS1)$ '

这将会给你一个像下面一样的展示：

user@Host:/path (master)$

注意点：

  • 根据系统在相应的文件添加这些修改并保存修改：~/.bashrc或者/ect/bashrc或者~/.bash_profile或~/profile。
  • 保存文件之后，运行source ~/.bashrc

使用PROMPT_COMMAND显示git分支

如果你在一个git仓库的文件夹里面，很容易显示你当前的分支名称。在~/.bashrc或者/etc/bashrc中添加下面的内容(这里是需要git的)：

function prompt_command {
    # 检查是否在一个git仓库里面
    if git status > /dev/null 2>&1; then
        # 只获取这个分支的名称
        export GIT_STATUS=$(git status | grep 'On branch' | cut -b 10-)
    else
        export GIT_STATUS=""
    fi
}

PROMPT_COMMAND=prompt_command

PS1="\$GIT_STATUS \u@\h:\w\$ "

如果我们在git仓库的目录里面，将会有下面的输出：

branch user@machine:~$

如果我们在一个普通的目录里面：

user@machine:~$

修改PS1提示

修改PS1，你只需要修改PS1shell变量的值。根据系统的不同，可以在~/.bashrc或/etc/bashrc中修改。PS1可以像任何文本一样修改：

PS1="hello "

在这些文本中，一些反斜杠的字符逃脱是可以支持的：

格式 含义
\a 一个ASCII闹铃字符(07)
\d “Weekday Month Date”格式的日期，像：”Tue May 26”
\D{format} 这个格式将会被传递给strftime(3)，结果插入提示字符串内。指定时间区域的空字符展示格式。这个括号是需要的。
\e 一个ASCII的逃脱字符(033)
\h 到第一个”.“的主机名
\H 主机名
\j 这个shell当前管理的job数量
\l shell的终端设备的基础地址名称
\n 新行
\r 回车
\s shell的名称，$0的基础名称(最后一个斜杠之后的部分)
\t 24小时的HH:MM:SS格式的当前时间
\T 12小时的HH:MM:SS格式的当前时间
\@ 12小时am\pm格式的当前时间
\A 24小时HH:MM格式的当前时间
\u 当前用户的用户名
\v bash的版本:例如2.00
\V bash的发布版本，假设补丁的信息：例如2.00.0
\w 当前的工作目录，有tilde的$HOME缩写
\W 当前工作目录的基础目录，有tilde的$HOME缩写
! 命令的历史数量
# 这个命令的命令数
\$ 如果有效的UID是0 返回#，否则返回$
\nnn* 与8进制数nnn对应的字符
反斜杠
[ 开始一个非打印字符序列，该序列可用于将终端控制序列嵌入提示中。
] 结束一个非打印字符序列

下面是一个例子，我们可以设置PS1：

PS1="\u@\h:\w\$"

这个将会输出：

user@machine:~$

显示前一个命令的返回状态和时间

有时候我们需要前一个命令返回值状态的视觉展示。下面的片段将会把这个放到PS1的开始位置。

需要注意每次一个新的PS1生成的时候，都要调用__stat()方法。否则它将保持.bashrc或.bash_配置文件最后一个命令的返回状态。

# -ANSI-COLOR-CODES- #
Color_Off="\033[0m" 
###-Regular-### 
Red="\033[0;31m" 
Green="\033[0;32m"
Yellow="\033[0;33m" 
####-Bold-####

function __stat() {
    if [ $? -eq 0 ]; then
        echo -en "$Green ✔ $Color_Off " 
    else
        echo -en "$Red ✘ $Color_Off " 
    fi
}

PS1='$(__stat)'
PS1+="[\t] " 
PS1+="\e[0;33m\u@\h\e[0m:\e[1;34m\w\e[0m \n$ "

export PS1

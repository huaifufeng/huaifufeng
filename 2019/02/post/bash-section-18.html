Sort

Option meaning
-u 每行输出都唯一

sort是一个Unix命令，用来按顺序对文件中的数据进行排序。

命令输出

sort用来排序一列行。

输入一个文件

sort file.txt

排序另个命令的输出

你可以排序任何输出命令的输出。在下面的例子中，会排序一列符合条件的文件：

find * -name pattern | sort

输出唯一

如果想要输出的行保持唯一，可以使用-u选项。

输出目录中文件的所有者：

ls -l | awk '{print $3}' | sort -u

数字排序

假设我们有下面的文件：

$ cat file 
10.Gryffindor 
4.Hogwarts 
2.Harry 
3.Dumbledore 
1.The sorting hat

想要按数字排序这些内容，使用-n选项：

sort -n file

这个命令会输出下面的内容：

1.The sorting hat
2.Harry
3.Dumbledore
4.Hogwarts
10.Gryffindor

想要倒序排列的话，使用-r选项：

sort -rn file

这将会输出下面的结果：

10.Gryffindor
4.Hogwarts
3.Dumbledore
2.Harry
1.The sorting hat

按key排序

假设我们有这样一个文件：

$ cat Hogwarts
Harry       Malfoy      Rowena       Helga
Gryffindor  Slytherin   Ravenclaw    Hufflepuff
Hermione    Goyle       Lockhart     Tonks
Ron         Snape       Olivander    Newt
Ron         Goyle       Flitwick     Sprout

想用某个列作为key来排序，可以使用k选项：

sort -k 2 Hogwarts

这将会把第2列作为排序的key，有下面的输出：

Ron        Goyle      Flitwick  Sprout
Hermione   Goyle      Lockhart  Tonks
Harry      Malfoy     Rowena    Helga
Gryffindor Slytherin  Ravenclaw Hufflepuff
Ron        Snape      Olivander Newt

如果我们在使用主key的基础上，想用再用第二个key来排序：

sort -k 2,2 -k 1,1 Hogwarts

这将会把第二列作为主key，然后将第一列作为第二个key：

Hermione   Goyle     Lockhart   Tonks
Ron        Goyle     Flitwick   Sprout
Harry      Malfoy    Rowena     Helga
Gryffindor Slytherin Ravenclaw  Hufflepuff
Ron        Snape     Olivander  Newt

当你使用一个以上的key排序时，对于每个-k选项都要指定排序结束。所以-k 1,1意味着以第一列开始排序，在第一列结束排序。

-t选项

在前面的例子中，文件有默认的分隔符tab。如果文件没有默认分隔符，我们需要使用-t选项指定分隔符。假设我们有下面的文件：

$ cat file
5.|Gryffindor 
4.|Hogwarts 
2.|Harry 
3.|Dumbledore 
1.|The sorting hat

使用第二列来排列这个文件：

$ sort -t "|" -k 2 file

将会有下面的输出：

3.|Dumbledore 
5.|Gryffindor 
2.|Harry 
4.|Hogwarts
1.|The sorting hat

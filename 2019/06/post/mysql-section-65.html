修改密码

1. Linux上面修改MySQL root的密码

修改MySQL root用户的密码：

**步骤1：**停止MySQL服务器

  • 在Ubuntu或Debian

    sudo /etc/init.d/mysql stop

  • 在CentOS，Fedora或Red Hat Enterprise Linux

    sudo /etc/init.d/mysqld stop

**步骤2：**不带权限系统开启MySQL服务器

sudo mysqld_safe --skip-grant-tables &

或者，如果mysqld_safe不可用

sudo mysqld --skip-grant-tables &

**步骤3：**连接MySQL服务器

mysql -u root

**步骤4：**为root用户设置新密码

Version>5.7

FLUSH PRIVILEGES;
ALTER USER 'root'@'localhost' IDENTIFIED BY 'new_password';
FLUSH PRIVILEGES;
exit;

Version<=5.7

FLUSH PRIVILEGES;
SET PASSWORD FOR 'root'@'localhost'=PASSWORD('new_password');
FLUSH PRIVILEGES;
exit;

注意：ALTER USER的语法是在MySQL5.7.6提供的。

**步骤5：**重启MySQL服务器

  • 在Ubuntu或Debian上

    sudo /etc/init.d/mysql stop
sudo /etc/init.d/mysql start

  • 在CentOS，Fedora或Red Hat Enterprise Linux

    sudo /etc/init.d/mysqld stop
sudo /etc/init.d/mysqld start

2. Windows修改MySQL root密码

当我们想要在Windows上面修改root的密码的时候，我们需要执行下面的步骤：

**步骤1：**使用下面的任意方式开启命令行终端

按下Ctrl+R或者到开始 > 运行，之后输入cmd，输入enter

**步骤2：**切换到MySQL按照的目录，

C:\> cd C:\mysql\bin

**步骤3：**开始mysql输入命令行

C:\mysql\bin> mysql -u root mysql

**步骤4：**使用下面的查询修改密码

mysql> SET PASSWORD FOR root@localhost=PASSWORD('my_new_password');

3. 处理过程

  1. 停止MySQL(mysqld)服务器/守护进程
  2. 使用—skip-grant-tables选项开启MySQL服务器，这样将不会要求输入密码：mysqld_safe —skip-grant-tables &
  3. 使用mysql -u root使用root用户连接到MySQL
  4. 修改密码
    • (5.7.6及之后)ALTER USER 'root'@'localhost' IDENTIFIED BY 'new-password'
    • (5.7.5及之前，或者MariaDB)SET PASSWORD FOR 'root'@'localhost'=PASSWORD('new-password'); flush privileges;quit;
  5. 重启MySQL服务器

注意：只有当你物理位于同一台机器时，才有效果。

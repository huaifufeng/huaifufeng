MySQL Client

参数 描述
-D —database=name 数据库的名称
—delimiter=str 设置语句的分隔符，默认是”;”
-e —execute=‘command’ 执行命令
-h —hostname=name 要连接的主机名称
-p —password=password 密码。注意：在-p和密码之间没有空格
-p (没有密码) 将会提示输入密码
-P —port=# 端口号
-s —slient 安静模式，产生更少的输出。使用-t作为列分隔符
-ss 和-s一样，但是忽略列名
-S —socket=path 当连接本地实例的时候，指定使用的socket(Unix)或pipe(Windows)
—skip-column-names 忽略列名
-u —user=name 用户名
-U —safe-updates —i-am-a-dummy 使用sql_safe_updates=ON变量登录。这在DELETE和UPDATE操作时，只允许指定Key的操作
-V —version 打印版本号并退出

1. 基本登录

使用命令行访问MySQL服务器：

mysql --user=username --password=pwd --host=hostname test_db

可以使用短命令模式：

mysql -u username -ppassword -h hostname test_db

通过省略password，MySQL会要求第一个输入密码。如果你在这里指定了password，那么客户端通给你一个不安全的警告：

mysql -u username -p -h hostname test_db

对于本地的连接，可以使用—socket来指定本地socket文件的地址：

mysql --user=username --password=pwd --host=localhost --socket=/path/to/mysqld.sock test_db

省略socket选项的话将会导致客户端尝试连接到本地计算机的服务器。服务器必须运行才能连接。

2. 执行命令

下面的例子展示了怎么执行存储在字符串或脚本文件中的命令，而不需要交互式的提示。这在需要使用shell连接数据库的时候很有用处。

执行字符串命令

$ mysql -uroot -proot test -e'select * from people'

+----+-------+--------+
| id | name  | gender |
+----+-------+--------+
|  1 | Kathy | f      |
|  2 | John  | m      |
+----+-------+--------+

想要使用tab分隔符格式化输出结果的话，使用—slient参数：

$ mysql -uroot -proot test -s -e'select * from people'

id      name    gender
1       Kathy   f
2       John    m

省略掉列的头部信息：

$ mysql -uroot -proot test -ss -e'select * from people'

执行脚本文件

$ mysql -uroot -proot test < my_script.sql

$ mysql -uroot -proot test -e'source my_script.sql'

输出到文件

$ mysql -uroot -proot test < my_scripte.sql > out.txt

$ mysql -uroot -proot test -s -e'select * from people' > out.txt

localhost 127.0.0.1

Localhost

特殊的DNS域名，分配给计算机本身使用。

不经网卡传输，不受网络防火墙和网卡相关设置的限制

访问localhost使用本机用户的权限去访问

127.0.0.1

通过网卡传输，依赖网卡，受网络防火墙和网卡设置限制

访问127.0.0.1 使用本机通过网络去访问本机，涉及到网络用户权限

访问本地程序时，使用localhost比较好，不占用网卡和网络资源
